Chrome-Network显示cookie

Posted on 2019-03-26

Chrome开发者工具中看不到网络请求的cookie信息

Chrome67引入饿一个新特性：site-isolation （站点隔离），也就是请求第三方站点数据时，如cookie等敏感信息会被隐藏显示为”Provisional headers are shown”

如果需要禁用站点隔离可以在chrome://flags配置

解决方法：
1.访问chrome://flags/#site-isolation-trial-opt-out

  1. 选择Opt-out

  2. 重启浏览器，完成。

css 个性化需要多主题

这个是很多年前的话题，主题修改整体颜色，第一想到的每个色块按着设计稿量取，再弄theme-red.css 或 theme-green.css 按照不用的需求加载不一样的css，这个效率及其低下。

  1. 开发第二套主题的人员不是第一套开发者
  2. 后期新增模块可能整体变动
  3. 整体变色文档保留不全
  4. 等等。。。。

这样就不是写一个可维护的代码，有什么更好的方式吗？例如通过一个颜色就能整体变色？需要克服的几点难题，

  1. 颜色，到底是color，还是background-color，还是border-color。css3提供currentColor提前子节点取到当前设置的color颜色。所以我们在根节点设置color，以下有几个问题需要考虑，
  2. 背景如果设置了currentColor，怎么设置color颜色，可以通过first-line来控制，
  3. svg可以通过fill来设置
  4. 有些颜色不是需要纯真的设置颜色，可以通过调整opacity来控制，css3强大的功能还可以根据hsl动态计算（本期未补充）
